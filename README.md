# E-Commerce-Data-Pipeline-for-a-Leading-Latam-E-commerce-Site

Python ETL project that:
- Extracts CSV and public holiday data
- Transforms datasets using pandas
- Loads analytical outputs for querying

### Tech stack
- Python
- Pandas
- Requests
- Pytest

### Structure
- src/: ETL logic
- queries/: SQL queries
- tests/: unit tests
